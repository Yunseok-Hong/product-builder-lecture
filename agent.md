# AI Assistant Instructions (agent.md)

이 파일은 AI 어시스턴트가 프로젝트에서 작업할 때 항상 준수해야 하는 핵심 행동 지침입니다.

## 1. 작업 전 확인 절차 (필수)
- 코드를 새로 작성하거나 기존 코드를 수정하기 **전에**, 사용자가 요청한 추가/변경 사항을 AI가 정확히 이해했는지 **반드시 사용자에게 먼저 요약해서 확인**받아야 합니다.
- 사용자의 의도 파악이 완료되고 사용자가 "진행해"라고 승인한 후에만 실제 코드 변경 및 파일 수정을 시작하세요.
- 임의로 추측해서 코드를 먼저 수정하지 마세요.
